Soalan Lazim

Yang anda mungkin mahu tahu

Staf saya berusia dan tak biasa guna telefon — boleh ke mereka guna?
Aplikasi staf hanya ada 3 tindakan utama — pilih kereta, pilih servis, tandakan siap. UI mudah + butang besar, latihan 1 jam sudah cukup. Kalau ada staf tersangkut kemudian, kami bimbing melalui WhatsApp — tiada kerja tambahan untuk anda.
Bagaimana dengan rekod ahli dan kad mata sedia ada?
Pemindahan data termasuk dalam servis. Berikan kami kad kertas, Excel atau senarai ahli sedia ada — kami susun dan import semuanya; baki mata sedia ada dikekalkan. Pelanggan tidak rugi apa-apa.
Pelanggan kena muat turun apps untuk lihat mata?
Tak perlu. Portal pelanggan ialah web apps (PWA) — hantar link melalui WhatsApp untuk buka; boleh ditambah ke skrin utama dan diguna seperti apps. Tiada App Store, tiada APK, mudah walau untuk pelanggan berusia.
Boleh saya cuba dulu sebelum buat keputusan?
Pelan Professional termasuk bulan pertama percuma — selepas setup siap, anda ada 30 hari untuk menilai. Tidak berpuas hati? Kami pulangkan penuh yuran setup. Anda hanya rugi masa pemindahan data, bukan duit.
Adakah data selamat? Boleh ke jualan atau data pelanggan bocor?
Data berada di Supabase di kawasan Singapura (mematuhi PDPA) dengan sandaran automatik harian. Hanya anda dan staf yang anda benarkan boleh log masuk — staf hanya lihat apa yang patut, tauke lihat semua. Walaupun syarikat kami hilang, anda boleh eksport 100% data anda.
Boleh saya edit pakej dan harga sendiri? Setiap cawangan caj berbeza.
Boleh. Panel admin membenarkan anda tambah atau edit pakej dan harga sendiri. Setiap cawangan boleh ada senarai harga sendiri — cucian sedan boleh RM 25 di Austin dan RM 22 di Skudai tanpa konflik.
Saya ada 2 cawangan — perlu beli 2 langganan?
Tak perlu. Professional sokong sehingga 2 cawangan; Enterprise sokong cawangan tanpa had. Setiap cawangan ada laporan staf, jualan dan stok sendiri — tetapi satu login meliputi semuanya.
Pejabat anda di mana di JB? Boleh ke saya hubungi kalau ada masalah?
Kami berada di Johor Bahru sendiri. Balasan WhatsApp segera, lawatan ke tapak bila perlu — ini perbezaan utama kami berbanding SaaS luar negara.
